FUTBIN Watcher

FUTBIN Watcher allows FIFA players to watch the price of the players of their choice in real-time.

Download

Click the Download ZIP File button above.

Usage

1. Specify the players you want to watch

  • Open the players_list.txt file.
  • Specify the URL of the player and your target price, separated by a space

Example:

http://www.futpc.com/player/3095/marco-reus 80000
http://www.futpc.com/player/3199/ramires 20000
http://www.futpc.com/player/3197/filipe-lu%C3%ADs 5000
http://www.futpc.com/player/3093/mesut-%C3%B6zil 4000

or 

http://www.futbin.com/player/3095/marco-reus 80000
http://www.futbin.com/player/3199/ramires 20000
http://www.futbin.com/player/3197/filipe-lu%C3%ADs 5000
http://www.futbin.com/player/3093/mesut-%C3%B6zil 4000

or

/player/3095/marco-reus 80000
/3199/ramires 20000
/3197/filipe-lu%C3%ADs 5000
/3093/mesut-%C3%B6zil 4000

All these work.

2. Run

  • Open the run.bat (Windows) or the run (Mac/Linux) file with a text editor (Notepad) for example.
  • Replace pc by your platform if needed (ps or xbox)
  • If you want the prices to be refreshed, specify a refresh delay; otherwise don't put anything
  • Save the file
  • Double click on run.bat (Windows) or run (Mac) like a program

Example of the run* files:

  • …futbin-watcher-x.x-jar-with-dependencies.jar pc 2: for PC, with the players list refreshing every 2 minutes
  • …futbin-watcher-x.x-jar-with-dependencies.jar xbox: for Xbox, with a single time execution
